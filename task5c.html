1. Solid Color

The simplest background — a single flat colour.

background-color: #1a1a2e;

2. Linear Gradient — Left to Right

Smoothly transitions between two colours horizontally.

background: linear-gradient(to right, #e96c6c, #ffcc00);

3. Linear Gradient — Diagonal, Multiple Stops

You can add as many colour stops as you like, and control the angle in degrees.

background: linear-gradient(135deg, #0f3460, #533483, #e94560);

4. Radial Gradient

Radiates outward from a point. circle at center defines the shape and origin.

background: radial-gradient(circle at center, #ffcc00, #e96c6c, #1a1a2e);

5. Image Background

cover scales the image to fill the container. center keeps it centred.

background: url('image2.jpg') center / cover no-repeat;

6. Image + Dark Overlay

Two backgrounds stacked. The gradient (top layer) darkens the photo so white text stays readable. This is the standard hero-section pattern.

background: linear-gradient(rgba(0,0,0,0.6), rgba(0,0,0,0.6)), url('image2.jpg');

7. Animated Gradient

The gradient is made 400% wide and an animation shifts the background-position back and forth, creating a flowing colour cycle.

animation: gradientShift 6s ease infinite;

8. RGBA — Semi-Transparent Color

The checkerboard behind this box bleeds through. The 4th value in rgba() controls opacity (0 = invisible, 1 = solid).

background-color: rgba(0, 87, 183, 0.4);
