CSS Properties: Lists, Links & Display

1. List Properties

list-style-type

Controls the marker (bullet or number) shown before each item.

  • disc (default ul bullet)
  • circle
  • square
  1. decimal (default ol)
  1. lower-alpha
  1. upper-roman
  • none (no marker)

list-style-position

Controls whether the bullet sits inside or outside the text block.

  • inside — bullet is inside the text box (wrapping text goes under bullet)
  • Second item to show wrapping

  • outside — bullet hangs in the left margin (default behaviour)
  • Second item to show wrapping

Custom styled list (no bullets)

Remove default bullets with list-style: none and style freely with borders/backgrounds.

2. Link Properties

Links have four states targeted with pseudo-class selectors. The order matters: :link → :visited → :hover → :active (remember LoVe HAte).

:link — unvisited (blue underline)

:hover — hover over me (red, no underline)

:active — click and hold me (orange)

Other common link properties

Property What it does
color Sets the link text colour
text-decoration underline (default) or none to remove it
background-color Highlight behind the link text
cursor pointer shows the hand icon (browsers do this by default on links)

3. The display Property

display controls how an element is placed in the page flow.

display: block

Each element takes the full width and starts on a new line. (<div>, <p>, <h1>–<h6> are block by default.)

display: inline

Elements sit side by side in a line. Width and height have no effect. (<span>, <a>, <strong> are inline by default.)

Sits inline like text, but you can set width, height, and full box-model properties. Great for buttons and cards.

display: none vs visibility: hidden

Both hide elements — but display: none removes it from the flow entirely (no space left behind), while visibility: hidden hides it but keeps its space.

Visible display:none — I am gone! Visible again — no gap

Visible visibility:hidden — space kept! Visible again — gap remains

Quick Reference

Value Line break? Width/Height? Use case
block Yes Yes Sections, paragraphs
inline No No Text spans, anchors
inline-block No Yes Buttons, badges
flex Container is block Yes Nav bars, card rows
grid Container is block Yes Page layouts, galleries
none — — Hide element completely
